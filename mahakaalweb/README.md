# Mahakaal Landing Page

Modern responsive single-page landing website for the Mahakaal mobile application. Built with React + Tailwind CSS.

**Branding:** Mahakaal by KrishnaTech Innovations

## Tech Stack

- React 18
- Vite
- Tailwind CSS

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) to view the site.

## Build

```bash
npm run build
```

## Preview Production Build

```bash
npm run preview
```
